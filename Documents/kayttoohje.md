Ennen ohjelman käynnistystä tulee asentaa tarvittavat riippuvuudet seuraavalla komennolla:
```bash
poetry install
```
Tämän jälkeen voit käynnistää pelin kirjoittamalla seuraavan komennon:
```bash
poetry run invoke start
```

### Ohjelman toiminta
